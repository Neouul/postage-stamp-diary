# 💌 우표 수집 다이어리 (Stamp Collector) 앱 기획서

## 1. 앱 컨셉 (Concept)

- **핵심 가치**: 일상의 순간을 '나만의 디지털 우표'로 만들어 간직하는 감성 기록 앱.
- **타겟 유저**: 다이어리 꾸미기를 좋아하는 사람, 수집벽이 있는 사람, 아날로그 감성을 선호하는 MZ세대.

## 2. 핵심 기능 상세

### ① 우표 프레임 카메라 (Stamp Camera)

- **실시간 뷰파인더**: 카메라 화면 위에 우표 특유의 '톱니바퀴 모양(Perforation)' 프레임을 오버레이로 표시.
- **다양한 스타일**: 빈티지, 현대적, 컬러풀 등 다양한 우표 테두리 디자인 선택 가능.
- **날짜/장소 각인**: 우표 하단이나 우인(Postmark) 디자인에 날짜와 위치 정보를 자동으로 기록.

### ② 나만의 우표첩 (Digital Stamp Album)

- **그리드 뷰**: 수집한 우표들을 실제 우표첩처럼 격자 형태로 나열.
- **카테고리 분류**: 여행, 음식, 일상 등 주제별로 우표첩을 생성하여 관리.
- **상세 보기**: 우표를 클릭하면 뒷면에 짧은 메모를 적거나 촬영 정보를 확인할 수 있는 'Flip' 애니메이션 적용.

### ③ 감성 통계 및 공유

- **수집 현황**: 이번 달에 몇 개의 우표를 모았는지 통계 제공.
- **이미지 내보내기**: 실제 다꾸에 활용할 수 있도록 흰색 배경이나 PNG 투명 배경으로 갤러리에 저장.

## 3. 기술 스택 및 구현 로직

### 🛠 기술 스택 (Android Native)

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose (애니메이션 및 선언형 UI 최적화)
- **Camera**: CameraX (안드로이드 공식 카메라 라이브러리)
- **Local DB**: Room (우표 정보, 메모, 경로 저장)
- **Image Handling**: Coil (이미지 로딩), Canvas API (우표 마스크 처리)

### 📸 우표 모양 이미지 처리 로직

1. **CameraX Capture**: 사각형 형태의 사진을 촬영합니다.
2. **Masking (Canvas API)**: 촬영된 사진 위에 우표 모양의 톱니바퀴 마스크를 씌워 `Bitmap`을 생성합니다.
3. **Save**: 마스킹된 결과물과 원본 사진을 구분하여 저장하거나, 마스킹 좌표만 DB에 저장합니다.

## 4. UI/UX 디자인 가이드

### 메인 컬러 (Primary Color)

- `#FDF5E6` (Old Lace): 빈티지한 종이 느낌의 배경색.
- `#8B4513` (Saddle Brown): 우드 톤의 따뜻한 포인트 컬러.

### 주요 화면 구조

- **Camera Screen**: 전체 화면 카메라 + 하단 셔터 버튼 + 상단 프레임 선택 슬라이더.
- **Album Screen**: `LazyVerticalGrid`를 사용한 우표 목록.
- **Detail Screen**: 우표 앞/뒷면 전환 (Card Flip Effect).

## 5. 향후 확장 아이디어

- **우인(Postmark) 스티커**: 우표 위에 찍히는 도장 디자인을 커스터마이징.
- **친구와 우편 교환**: 내가 만든 우표를 친구의 앱 내 우체통으로 보내는 소셜 기능.
- **AR 뷰어**: 실제 다이어리 위에 폰을 갖다 대면 내가 모은 디지털 우표가 증강현실로 나타나는 기능.
